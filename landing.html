Loan Eligibility Project

Streamline the loan eligibility assessment process through the
use of statistical analysis and machine learning algorithms.

About this project

The data set is from a data science contest hosted by AnalyticsVidhya.com. It will be demonstrated in this project how one business process can be optimized by the application of data analytics. Inferences backed by statistical analysis will be explained in a simplified context by use of data mining and data visualization.
